# Космический Рывок (Space Dash) v4.0

Аркадная космическая игра с сюжетом в стиле визуальной новеллы для Яндекс.Игры.

## Описание

Игрок управляет космическим кораблём в экспедиции к системе Кеплер-442. По пути нужно уклоняться от астероидов, стрелять по врагам, собирать кристаллы и достичь финальной цели на 1200 м. Сюжет разворачивается через кат-сцены с персонажами в стиле визуальной новеллы.

## Особенности v4.0

| Фича | Описание |
|------|----------|
| Визуальная новелла | Кат-сцены с 5 персонажами (Диспетчер, Капитан, Инженер, ИИ-голограмма, Инопланетянин) |
| 5 локаций | Земля → Орбита → Открытый космос → Туманность → Зона боя |
| 6 скинов кораблей | Стрела, Феникс, Золотой, Фантом, Изумруд, Ледяной |
| Механика стрельбы | Пробел / кнопка (стоимость 10 кристаллов) |
| Мини-босс на 700 м | Кат-сцена + волна вражеских кораблей |
| Финал на 1200 м | Посадка на планету, установка базы, атака захватчиков |
| 5 улучшений | Скорость, Щит, Магнит, Удача, Броня (до 5 уровней) |
| 3 музыкальных трека | Атмосферная музыка для меню, геймплея и босса |
| 7 звуковых эффектов | Выстрел, взрыв, сбор, щит, удар, бонус, клик |
| Плавные переходы | Fade-анимация при смене локаций |
| Адаптивность | Десктоп + мобильные (тач-управление, стрелки на экране) |
| Yandex Games SDK | Реклама (fullscreen + rewarded), лидерборды, облачное сохранение |

## Монетизация

| Тип рекламы | Когда показывается |

## Публикация на Яндекс.Игры

1. Перейдите на https://games.yandex.ru/console
2. Нажмите «Добавить приложение» и загрузите `space-dash-game.zip`
3. Заполните информацию:
   - Название: Космический Рывок
   - Жанр: Аркады
   - Устройства: Десктоп + Мобильные
   - Ориентация (мобильные): Портретная
   - Языки: Русский, Английский
   - Возрастной рейтинг: 6+
4. Создайте лидерборд с именем `score` (числовой, по убыванию)
5. Подключите монетизацию РСЯ на вкладке «Реклама»
6. Отправьте на модерацию (3-5 рабочих дней)

## Управление

| Платформа | Действие | Управление |
|---|---|---|
| Десктоп | Движение | ← → / A D |
| Десктоп | Стрельба | Пробел |
| Десктоп | Пауза | Esc / P |
| Мобильные | Движение | Стрелки на экране / Свайп |
| Мобильные | Стрельба | Кнопка 🎯 |

## Технические детали

- Чистый HTML5 Canvas + JavaScript (без фреймворков)
- Все ассеты встроены (нет внешних зависимостей кроме Yandex SDK)
- localStorage для локального сохранения + Yandex Cloud Save
- RequestAnimationFrame для плавной анимации
- Адаптивный canvas (resize)
- Размер архива: ~13 МБ (лимит: 100 МБ)
