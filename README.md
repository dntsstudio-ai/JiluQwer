# Космический Рывок (Space Dash) v2.0

Аркадная HTML5-игра для платформы Яндекс.Игры с сюжетом, сменой локаций и мини-боссами.

## Описание игры

Игрок управляет космическим кораблём в экспедиции к краю обитаемого космоса. Уклоняйся от астероидов, собирай кристаллы, улучшай корабль и сражайся с инопланетными преследователями! Игра имеет сюжетные кат-сцены, 5 уникальных локаций, систему улучшений и скинов.

## Особенности v2.0

- **Сюжетные кат-сцены** — история в стиле визуальной новеллы
- **5 уникальных локаций** — Земля → Орбита → Открытый космос → Туманность → Погоня!
- **Мини-босс** — на 200 м начинается погоня инопланетных кораблей, стреляющих в игрока
- **Система улучшений** — 5 параметров корабля (скорость, щит, магнит, удача, броня)
- **6 скинов корабля** — покупаются за кристаллы
- **Возрождение** — за просмотр рекламы или за 50 кристаллов
- **Плавный баланс сложности** — 5 минут до максимальной скорости
- **Полная интеграция Yandex Games SDK** — реклама, лидерборды, облачное сохранение
- **Локализация**: русский и английский (автоопределение через SDK)
- **Адаптивный дизайн** (десктоп + мобильные устройства)
- **Управление**: клавиатура (← →, A/D) и тач/свайп
- **Размер** < 20 КБ (требование — до 100 МБ)

## Как опубликовать на Яндекс.Игры

### Шаг 1. Регистрация
1. Перейдите на https://games.yandex.ru/console
2. Войдите с аккаунтом Яндекс
3. Примите условия разработчика

### Шаг 2. Загрузка игры
1. Нажмите «Добавить игру»
2. Загрузите файл `space-dash-game.zip`
3. Заполните информацию:
   - **Название**: Космический Рывок
   - **Описание**: Управляй космическим кораблём, уклоняйся от астероидов и собирай кристаллы! Бесконечная аркада с нарастающей сложностью.
   - **Жанр**: Аркады
   - **Устройства**: Десктоп + Мобильные
   - **Ориентация (мобильные)**: Альбомная (landscape)
   - **Языки**: Русский, Английский
   - **Возрастной рейтинг**: 6+

### Шаг 3. Настройка рекламы
1. Перейдите на вкладку «Реклама»
2. Подключите монетизацию РСЯ
3. Заполните реквизиты для выплат

### Шаг 4. Настройка лидерборда
1. В консоли разработчика создайте лидерборд с именем `score`
2. Тип: числовой, сортировка по убыванию

### Шаг 5. Модерация
1. Отправьте игру на модерацию
2. Ожидайте 3-5 рабочих дней
3. После одобрения нажмите «Опубликовать»

## Монетизация

Игра использует три типа рекламы:
- **Полноэкранная реклама** — показывается при старте игры и после Game Over (не чаще 1 раза в 3 минуты)
- **Rewarded Video** — игрок может посмотреть рекламу для возрождения после смерти
- **Sticky Banner** — настраивается в консоли разработчика

## Технические детали

- Чистый HTML5 + Canvas + JavaScript (без фреймворков)
- Размер архива: ~12 КБ
- Нет внешних зависимостей
- Совместимость: все современные браузеры

## Советы по увеличению дохода

1. Регулярно обновляйте игру (новые бонусы, уровни)
2. Следите за рейтингом (должен быть выше 30)
3. Добавьте больше поводов для просмотра rewarded video
4. Оптимизируйте частоту показа рекламы
